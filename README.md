# CV

A personal CV website built with Jekyll and hosted on GitHub Pages. The site is automatically deployed through GitHub's
built-in Pages integration, and includes a GitHub Actions workflow that automatically generates and publishes a PDF
version of the CV alongside the web version.

**Live Site:** https://svalasovich.github.io/cv/

## Features

- Static site generation with Jekyll
- Automatic deployment to GitHub Pages
- Automated PDF generation via GitHub Actions workflow
